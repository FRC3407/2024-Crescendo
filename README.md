# 2024 Crescendo

## Description

The FRC 2024 competition code for 3407 Wildcards

## Prerequisites

* SPARK MAX Firmware v1.6.2 - Adds features that are required for swerve
* REVLib v2023.1.2 - Includes APIs for the new firmware features

## Configuration

It is possible that this project will not work for robots right out of the box. Various things like the CAN IDs, PIDF gains, chassis configuration, etc. must be determined and adjusted in the `Constants.java` file.
